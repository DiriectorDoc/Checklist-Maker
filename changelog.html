v0.2_Beta

This is the first Beta update. It's been more than a week since the last alpha update, but now, we're flipping the switch; this project is now in Beta, not that anything has changed.

So, what's the deal with these now-called Beta versions? All the features initially intended to be implemented have been implemented. There will still be content updates here and there, but most of the updates from here on out will mainly be UX improvements and bug fixes.

Changes

Major

  • Began hosting this project at diriectordoc.github.io/Checklist-Maker
  • Altered version naming convention: "v0.1.X_alpha" → "v0.X_Beta"
  • Tweaked the scrolling that happens after adding/removing an item from the list
  • Input fields will now save its value if you click off of it
  • Pressing ↵ Enter still works, but now there won't be numerous input fields all over the place.

  • If an input field is blank on submission, the item will default to the input field's placeholder
